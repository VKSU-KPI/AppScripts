# Google Sheets Automation Scripts

This repository contains Google Apps Script code designed to automate and enhance our daily workflows in Google Sheets. It serves as a centralized hub for all the custom scripts we use to manage data, handle interactive dashboards, and streamline our spreadsheet operations.

The codebase includes various functions for dynamic data validation, automated sheet generation from specific templates, and custom UI triggers within our spreadsheets. By keeping these scripts version-controlled here, we can easily track changes, collaborate on improvements, and ensure our spreadsheet tools remain reliable and up-to-date.

To deploy any of these tools, you need to open the target Google Sheet, navigate to the Apps Script editor through the Extensions menu, and paste the relevant code. Certain functions, especially those relying on simple triggers like `onEdit`, will run automatically once saved, while others may require manual assignment to clickable drawings or buttons within the sheet itself.
